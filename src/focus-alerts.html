Focusable Element Alerts Reference

This page lists and explains the focus-related alerts used in this extension, based on the ANDI Focusable Element Alerts.

Tabindex Alert
Indicates that the element has a tabindex attribute, which affects its focusability and tab order.
Details: Elements with tabindex="-1" are not tabbable but can be focused programmatically. Elements with tabindex="0" are tabbable in document order. Positive values set a custom tab order, which can be confusing for keyboard users.
Not Tabbable Alert
Indicates that the element is not tabbable, meaning it cannot be reached using the Tab key. This may be due to tabindex="-1", being hidden, or other reasons.
ARIA Hidden Alert
Indicates that the element or one of its ancestors has aria-hidden="true", which removes it from the accessibility tree and prevents screen readers from interacting with it.
Invisible Element Alert
Indicates that the element is visually hidden (e.g., display:none, visibility:hidden, or opacity:0), which may affect its focusability and accessibility.
Disabled Element Alert
Indicates that the element is disabled (e.g., disabled attribute on form controls), making it unfocusable and unusable for keyboard users.
Role Presentation Alert
Indicates that the element has role="presentation" or role="none", which removes its semantic meaning from the accessibility tree and may affect its focusability.
Focus Event Alert
Indicates that the element has JavaScript event handlers related to focus (e.g., onfocus, onblur), which may affect its behavior when focused.
Autofocus Alert
Indicates that the element has the autofocus attribute, which causes it to receive focus automatically when the page loads.
Focusable Descendant Alert
Indicates that the element contains a descendant that is focusable, which may affect navigation and accessibility.
Custom Focusable Alert
Indicates that the element is made focusable through custom scripting or ARIA attributes, rather than native HTML properties.

Alert descriptions adapted from the ANDI Focusable Element Alerts by the Social Security Administration.
